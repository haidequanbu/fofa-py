# fofa-py

**fofa的python3版本的请求接口**

### 包依赖

1.python3.8

2.requests 2.27.1

**测试环境：macbook air 2020 M1**

### 使用

1.将fofa.py拷贝到自己的文件夹下

2.参考demo.py对对应接口查询即可
